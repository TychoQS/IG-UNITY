# IG-UNITY
### Práctica optativa - Informática Gráfica (ULPGC)
**Curso 2025/2026**

## La idea
Este proyecto nace inspirado por la gran cantidad de titulares diarios sobre retenciones y accidentes en la **GC-1**. Busca reflejar, con un toque de humor, esa situación cotidiana en nuestras carreteras.

## El juego
La premisa es sencilla: el objetivo es circular con el vehículo intentando no tener percances de ningún tipo por el camino.

## Controles

| Acción | Teclas |
| :--- | :--- |
| **Moverse** | `W`, `A`, `S`, `D` o `Flechas de dirección` |

## Nota importante
Este juego es una parodia con fines académicos. Cualquier parecido con el tráfico real de la isla es pura coincidencia.
